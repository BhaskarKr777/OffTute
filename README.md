# OffTute

OffTute is an open-source, offline personal AI tutor powered by the Mistral
language model. It is designed to run locally on decent laptops without any
paid APIs, helping students learn by reading on-screen content, understanding
voice commands, and explaining topics in simple, human-friendly language.

## Key Features
- Offline AI tutor using Mistral via Ollama
- Screen reading using OCR
- Voice-based interaction (multilingual)
- Simple explanations with real-world examples
- Runs on CPU-only systems
- Fully open-source (MIT License)

## Tech Stack
- LLM: Mistral (Ollama)
- OCR: Tesseract
- Speech-to-Text: Whisper (offline)
- Text-to-Speech: Piper TTS
- Language: Python
